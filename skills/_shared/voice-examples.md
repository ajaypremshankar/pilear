## How to use in the pipeline

1. Read 2–3 examples closest to the post type (career, technical, personal).
2. Match **rhythm**: sentence length variety, one-line paragraphs, direct address.
3. Do **not** transplant metaphors unless they fit the topic.
4. User's reflection phrasing always overrides these examples.
