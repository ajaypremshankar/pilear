# NOTES.md Format (learning root)

`NOTES.md` lives at `<learning-root>/NOTES.md` (workspace root where `pi` was started).

Scratchpad for teaching preferences and working notes that apply across all topics.

## Examples

```md
# Teaching preferences

- Prefer scenario quizzes over abstract definitions
- Push hard on production failure modes
- Do not suggest Reddit communities — I prefer papers and postmortems only
- Session length: ~30 minutes max
```

## Rules

- Read at session start when present (`deep-dive`, `recall`, `discover`).
- Append when the user states a durable preference — do not overwrite without asking.
- Do not put topic-specific mission or resources here — those belong under `<domain>/<subject>/`.
